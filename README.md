# Astraea Planetarium Café

This is a fictional planetarium café website created s part of a college course in interface design using CSS.
Astraea is inspired by the idea of bringing the universe closer to people.

## 🌌 Concept

The concept is to provide a relaxing space where people can experience the beauty of the universe in their everyday lives.
Customers can enjoy planetarium shows in a cozy café setting, and even bring that experience home through projector products.

## 🎯 Features

- Structuring a visually consistent UI
- Designing with a clear concept and theme

## 🛠 Tech Stack

- HTML
- CSS
- JavaScript
